# 桌面宠物技能系统使用说明

## 功能概述

技能系统允许从 [skills.sh](https://skills.sh) 生态安装技能，扩展桌面宠物的能力。每个技能包含一组工具调用和使用说明，让宠物能够执行新的任务。

## 使用方法

### 1. 进入技能管理

右键点击宠物 → **设置** → 向下滚动找到 **技能管理** 卡片。

### 2. 搜索技能

在搜索框中输入关键词（如 `vercel`、`git`、`翻译` 等），点击"搜索"按钮或按回车。系统会调用 `npx skills find` 命令搜索可用的技能包。

### 3. 安装技能

在搜索结果中找到想要安装的技能，点击"安装"按钮。系统会：
- 执行 `npx skills add <package>` 命令下载技能
- 解析技能的 `SKILL.md` 文件
- 保存技能信息到配置目录

### 4. 管理已安装技能

在"已安装的技能"列表中可以：
- **启用/禁用**：切换技能开关，禁用的技能不会被 LLM 使用
- **删除**：卸载技能
- **查看详情**：点击技能名称展开，查看技能描述、作者、版本和使用说明

### 5. 使用技能

技能安装并启用后，在与宠物对话时，LLM 会自动：
- 在系统提示中获取已启用技能的描述
- 根据技能的使用说明理解如何使用该技能
- 当用户请求相关任务时，自动调用技能提供的功能

## 技能存储位置

技能文件存储在以下位置：
- **配置文件**: `~/.config/desktop-pet/skills.json`
- **技能文件**: `~/.config/desktop-pet/skills/.agents/skills/`

## 技能格式

每个技能包含一个 `SKILL.md` 文件，格式如下：

```markdown
---
name: 技能名称
description: 技能描述
author: 作者名称
version: 1.0.0
---

这里是技能的使用说明，告诉 LLM 如何使用这个技能。

当用户询问相关问题时，LLM 会根据这里的内容理解如何处理。
```

## 通过聊天安装技能

你也可以直接在聊天中让宠物帮你安装技能：

1. 用户：帮我找一个能部署到 Vercel 的技能
2. 宠物会调用技能搜索工具，列出找到的技能
3. 用户：安装第一个技能
4. 宠物会调用安装工具完成安装

## 注意事项

1. **Node.js 要求**：技能安装需要系统已安装 Node.js 和 npm
2. **网络连接**：搜索和安装技能需要网络连接
3. **安全提示**：安装第三方技能前请确认来源可靠，技能可能包含可执行代码

## 故障排除

### 搜索失败
- 检查网络连接
- 确认已安装 Node.js 和 npm
- 尝试在终端手动执行 `npx skills find <query>` 测试

### 安装失败
- 检查磁盘空间
- 确认有写入配置目录的权限
- 查看错误信息中的具体原因

### 技能不生效
- 确认技能已启用（开关为绿色"启用"状态）
- 重启应用或刷新聊天窗口
- 检查技能的使用说明是否正确
